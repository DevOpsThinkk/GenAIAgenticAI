# Generative AI and Agentic AI Course

A comprehensive course on **Generative AI** and **Agentic AI** delivered at IIT Patna. This course covers the fundamental concepts, implementation techniques, and practical applications of generative models and autonomous AI agents.

## 📋 Course Overview

This course is designed to provide students with a deep understanding of:
- **Generative AI**: Building and training models that generate new content (text, images, code, etc.)
- **Agentic AI**: Creating autonomous agents capable of reasoning, planning, and decision-making
- Practical implementations using Python and modern AI frameworks
- Real-world applications and best practices


## 🎓 Lectures

The course consists of multiple lectures covering progressive topics:

- **Lecture 3**: Introduction to Generative Models
- **Lecture 4**: Advanced Generative AI Techniques
- **Lecture 5**: Agentic AI and Autonomous Systems

Each lecture includes:
- Detailed explanations
- Code implementations
- Practical examples
- Theoretical foundations

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- Basic understanding of machine learning concepts
- Familiarity with Python programming

### Installation

1. Clone or download this repository
2. Install required dependencies:
```bash
pip install -r requirements.txt
```


## 📝 Course Content

### Generative AI
- Transformer architectures and attention mechanisms
- Language models and text generation
- Image generation and diffusion models
- Prompt engineering and fine-tuning
- Ethical considerations in generative AI

### Agentic AI
- Agent architecture and design patterns
- Decision-making and reasoning systems
- Planning and task decomposition
- Tool use and external integration
- Multi-agent systems and collaboration

## 🛠️ Technologies & Tools

- **Python**: Primary programming language
- **AI Frameworks**: PyTorch, TensorFlow, or similar
- **Libraries**: NumPy, Pandas, Scikit-learn
- **API Integration**: LLM APIs and external services

## 📚 How to Use This Repository

1. **Follow lectures sequentially** starting from Lecture 3
2. **Run the code examples** in each lecture file
3. **Complete practice exercises** in the `Practice/` folder
4. **Experiment in roughbook.py** for deeper exploration
5. **Review and modify code** to understand concepts better

- Apply generative AI and agentic AI to real-world problems
